# Sistem Informasi Akademik

Sistem informasi akademik untuk manajemen sekolah yang mencakup:
- Manajemen Data Guru
- Absensi Siswa
- Jadwal Pelajaran
- Manajemen Kelas
- Manajemen Mata Pelajaran

## Teknologi yang Digunakan

- Laravel 10
- Livewire 3
- Alpine.js
- Tailwind CSS


## Fitur

- Dashboard overview
- Manajemen data guru
- Absensi siswa
- Jadwal pelajaran
- Manajemen kelas
- Manajemen mata pelajaran
